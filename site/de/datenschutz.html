Datenschutz

Kurzfassung: Diese Projektseite enthält keine Analyse- oder Werbeskripte. Die Desktop-App läuft lokal. Daten verlassen dein Gerät nur, wenn du einen entfernten Anbieter, Connector, MCP-Server oder ein netzwerkfähiges Werkzeug konfigurierst und verwendest.

Projektwebsite

Diese statische Seite setzt keine eigenen Cookies, erstellt keine Benutzerkonten und lädt keine externen Schriftarten, Analyse-, Werbe-, Pixel- oder Session-Replay-Skripte. Sie wird über GitHub Pages bereitgestellt. GitHub kann technische Anfragedaten wie IP-Adresse und Browserinformationen gemäß der GitHub-Datenschutzerklärung verarbeiten.

Daten der Desktop-App

Open Cowork speichert Anwendungszustand, Aufgabenverlauf, Audit-Informationen und ausgewählten Arbeitskontext lokal auf dem Computer. Die App betreibt kein Open-Cowork-Cloudkonto und keinen eigenen Synchronisierungsdienst.

Modellanbieter und verbundene Werkzeuge

Wenn du einen entfernten Modellanbieter, Connector, MCP-Server, Webhook oder ein netzwerkfähiges Tool verwendest, sendet die App die für deine angeforderte Aktion benötigten Informationen an diesen Endpunkt. Empfänger und Datenschutzbedingungen hängen von deiner Konfiguration ab. Mit Ollama können Modellanfragen lokal verarbeitet werden.

Zugangsdaten

Unter Windows werden unterstützte Anbieter-Schlüssel und Connector-Secrets über die Windows-Anmeldeinformationsverwaltung gespeichert. Sensible Werte werden aus dem normal gespeicherten Frontend-Zustand entfernt. Audit- und Diagnosepfade im Backend redigieren übliche Secret-Felder und Tokenformate.

Telemetrie

Die aktuelle Version enthält keinen implementierten Telemetrie-Sender; die Telemetrie-Einstellung ist standardmäßig deaktiviert. Vor einer zukünftigen Einführung projektbetriebener Analyse- oder Telemetrieübertragung muss dieser Hinweis aktualisiert werden.

Lokale Berechtigungen

Open Cowork kann mit lokalen Dateien, Terminals, Prozessen und verbundenen Werkzeugen arbeiten. Prüfe Arbeitsbereichszugriff, Berechtigungseinstellungen, Freigabeaufforderungen und Drittanbieter-Konfigurationen, bevor du die App mit sensiblen Inhalten verwendest.

Sicherheitsmeldungen

Veröffentliche keine Zugangsdaten oder vertraulichen Inhalte in einem öffentlichen Issue. Melde Sicherheitslücken privat über GitHub Security Advisories.

Fragen

Für nicht sensible Datenschutzfragen kannst du eine Datenschutzfrage eröffnen. Dieser Hinweis beschreibt das Open-Source-Projekt und ersetzt nicht die Bedingungen verbundener Modellanbieter oder Dienste.
